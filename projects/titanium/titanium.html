Characterization of Arconic Titanium Powders for 3D Printed Parts

For our Materials Science and Engineering capstone project, Marissa, Sabrina, David, and I partnered with John Barnes, VP for Advanced Manufacturing and Strategy of Arconic. We set out to characterize the Class C titanium powders to its possible use in the additive manufacturing industry and analyze Class A2 powders of different reuse iterations to understand the effect of reuse on the size distribution and shape of the titanium powder. Our analysis included characterization techniques ranging from powder rheometry to scanning electron microscopy (SEM) to machine vision.

My work on the project dealt almost entirely with the imaging and image processing. We took hundreds of SEM images, which were then analyzed in ImageJ to extract the particle sizes. Among other processing techniques, automatic and manual watershed segmentation was performed to ensure ImageJ properly identified the powders, i.e. did not process two particles as being one if they were close together. At this point, the Class A2, Class A2 reused, and Class B powders were analyzed by ImageJ. An extra step was performed on the Class C powders, breaking the image analysis into two cases. In Case 1, the images of the Class C powders were manually watershed, such that each particle and satellites were measured separately, regardless of whether the satellite was attached to a larger powder. In Case 2, the automatic watershed was corrected, such that ImageJ would measure each particle with its satellites as one particle.

a. SEM image of Class C powder (scale bar = 500 microns); b.and c. Case 1 and 2 ImageJ analysis, respectively, of the same image

Particle size analysis in ImageJ, followed by size distribution analysis in R showed the significant impact the satellites have on the Class C powders.

d. and e. Histograms of Class C powders with Case 1 and 2 analyses, respectively

The next step was to introduce a new technique for image analysis: machine vision. Machine vision is a computer program, written by a Carnegie Mellon University MSE graduate student, that utilizes machine learning to quantitatively visualize similarities between powders, which we used to analyze the reused powders. Briefly, the program is “trained” on a series of images of one class of powder to learn the characteristic size distribution and shape factor. A new set of images of the same class is then run through the program, which recognizes patterns and determines if the new set of images belongs with the original. In the case of comparison of reused powders, the algorithm was trained on images of Class A2 powders and then tested on the images Class A2 powders that had been reused four times to see if the program categorized the two sets of powders differently. As can be seen in the principal component analysis (PCA) map, the A2 and reused A2 powders were in fact characterized differently.

PCA map clusters images based on features such as size and shape

Based on these findings, we recommended to Arconic that the Class C powders be subclassified into groups with smaller size distributions. These powders could then be used in the 3D printing of porous components, such as those used in catalysis, as the porosity could be better controlled. Machine vision can be further used to understand the relationship between different iterations of reuse.
